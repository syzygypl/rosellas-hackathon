# Idea Lab Investor Pitch Video Scenario

## Feature Analysis

Idea Lab is an AI-assisted invention workspace for teams facing technical trade-offs. A user describes a problem in natural language, and the app turns it into a structured TRIZ problem-solving run.

Core features shown by the current app:

- Natural-language chat intake with short clarifying questions.
- Example starters for real engineering trade-offs.
- TRIZ-grounded solving through parameter search, contradiction matrix lookup, and inventive principle retrieval.
- Reusable solution cards with a plain-language contradiction, directions, next steps, full report, technical details, and reasoning trail.
- Hybrid engine: LLM agent when configured, deterministic TRIZ pipeline as fallback.
- Session traceability through copyable session ids and optional Langfuse tracing.

## Key Message

Idea Lab does not replace deterministic engineering logic with a generic chatbot. It wraps deterministic TRIZ tools in an LLM-driven conversation so users can describe real problems naturally while the system preserves structured retrieval, matrix lookup, traceability, and reusable outputs.
